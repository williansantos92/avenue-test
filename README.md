## Getting Started

Clone project:

```git
git clone <https://github.com/williansantos92/avenue-test.git>
```

Install dependencies

```js
npm i;
```

Run all tests locally

```js
npm run test:all
```

Run Ui tests locally

```js
npm run test:ui
```

Run Api tests locally

```js
npm run test:api
```
